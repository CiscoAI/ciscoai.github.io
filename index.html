Why Cisco Kubeflow Starter Pack

The Cisco Kubeflow Starter Pack is a set of end-to-end applications that help Cisco customers to manage a portable, distributed, and scalable machine learning system on on-premise Kubernetes clusters running on Cisco servers. Our mission is to help each of our customers to reduce the friction in managing ML infrastructure by eliminating the complex steps in build-train-deploy machine learning workflow.

We have been intensively developing and leading Kubeflow on-premise efforts in the community, and we understand how non-trivial on-premise deployments can get. Therefore, we developed a Cisco Kubeflow Starter Pack to help Cisco customers to run Kubeflow easily on Cisco infrastructure namely the UCS and HyperFlex Kubernetes cluster.
